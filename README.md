# Task Manager Application

A full-stack Task Management Application built using:

- React.js
- Node.js
- Express.js
- MongoDB Atlas
- JWT Authentication

## Features

- User Login Authentication
- Create Tasks
- View Tasks
- Delete Tasks
- Protected Routes using JWT
- MongoDB Database Integration

## Tech Stack

Frontend:
- React
- Axios

Backend:
- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT

## Run Project

Backend:

npm install
npm run dev

Frontend:

npm install
npm start
